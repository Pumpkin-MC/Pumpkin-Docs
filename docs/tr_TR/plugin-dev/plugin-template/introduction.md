# Örnek Bir Eklenti Yazma

Bu eğitim, yeni bir eklenti oluşturmanız için size rehberlik eder.

Bu eğitim, Rust ve komut satırı konusunda biraz deneyiminiz olduğunu varsayar.

Bu eğitimde anlatılan konular:

- Katılma mesajını değiştirme
- Taş-kağıt-makas komutu oluşturma
- Eklenti verilerini eklentinin dizinine kaydetme (yakında)
