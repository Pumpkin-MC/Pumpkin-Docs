# Pumpkin 插件开发

::: warning 警告
Pumpkin 插件 API 仍处于非常早期的开发阶段，可能随时会发生变化。
如果您遇到任何问题，请在我们的 [Discord](https://discord.gg/aaNuD6rFEe) 上联系我们。
:::

Pumpkin 插件与服务器软件在很深的层次上集成，允许实现其他服务器软件无法实现的许多功能。

Pumpkin 插件 API 在很多方面受到 Spigot/Bukkit 插件 API 的启发，因此如果您有这些 API 的使用经验并且熟悉 Rust 开发，您应该能够相对轻松地为 Pumpkin 编写插件。 :smile:
