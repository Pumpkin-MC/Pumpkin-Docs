# Pumpkin Plugin‑Entwicklung

::: warning
Die Pumpkin Plugin API befindet sich noch in einem sehr frühen Entwicklungsstadium und kann sich jederzeit ändern.
Wenn du auf Probleme stößt, melde dich bitte auf unserem [Discord‑Server](https://discord.gg/aaNuD6rFEe).
:::

Pumpkin‑Plugins integrieren sich sehr tief in die Server‑Software und ermöglichen dadurch viele Dinge, die auf anderer Serversoftware nicht möglich wären.

Die Pumpkin Plugin API orientiert sich in vielen Bereichen an der Spigot/Bukkit Plugin API. Wenn du damit bereits Erfahrung hast und Rust beherrschst, solltest du relativ einfach Plugins für Pumpkin schreiben können. :smile:
